# Tracker Lokasi

Tools Lacak Lokasi Seseorang
## Instalasi

### Prasyarat
- Node.js
- cloudflared

### Langkah

1. Clone repo ini
   
   git clone https://github.com/hylix2234/Tracker-Lokasi.git
   cd Tracker-Lokasi

2. Install dependencies
   
   npm install

3. Jalankan server
   
   bash start.sh

## Teknologi
- Node.js + Express
- Cloudflare Tunnel
- HTML + CSS + JavaScript
